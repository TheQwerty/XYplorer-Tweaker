XYplorer-Tweaker
================

An attempt to document and create a script to improve finding and modifying XYplorer's INI tweaks.



To be added as a contributor please [contact me](http://www.xyplorer.com/xyfc/ucp.php?i=pm&mode=compose&u=438 "Send message to TheQwerty on XYplorer's forums.") on XY's forums with a mention of this repository and a link to your GitHub profile.
